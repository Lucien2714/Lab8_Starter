### Graceful Degradation 
Graceful degradation means building an app so it still provides a usable experience when some features are unavailable. Service workers support graceful degradation by caching important files and data, allowing the site to keep working when the network is slow or offline. If the service worker is not supported or fails to register, the app should still fall back to normal network requests and remain usable online.
